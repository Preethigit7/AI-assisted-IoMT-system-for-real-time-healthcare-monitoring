# AI-assisted-IoMT-system-for-real-time-healthcare-monitoring
Developed an AI-assisted IoMT system using ESP32 and sensors for real-time data acquisition. Processed data in Python using NumPy and Pandas, and built a basic ML model for anomaly detection. Integrated the model and created a Streamlit dashboard for monitoring. Used VS Code; handled debugging, connectivity, and system optimization.

## 📄 Abstract  
This project presents the development of an AI-assisted Internet of Medical Things (IoMT)-based healthcare monitoring system designed to support advanced cancer therapy. 
The system integrates pH-sensitive chitosan-coated gold nanoparticles (CS-GNPs) with real-time physiological data collected through IoT sensors using an ESP32 microcontroller.
The chitosan coating enhances biocompatibility and enables targeted drug delivery, while its pH-responsive behavior facilitates controlled drug release in the acidic tumor microenvironment.

Real-time sensor data is processed using Python and visualized through a Streamlit dashboard. A threshold-based anomaly detection system identifies irregular patterns, 
enabling intelligent monitoring and timely intervention.

## 💡 Key Features  
- Real-time data collection using ESP32 and sensors  
- AI-assisted anomaly detection  
- Streamlit dashboard for visualization  
- Integration of IoT with healthcare monitoring  

## 🛠️ Tech Stack  
- Python  
- ESP32  
- IoT Sensors  
- Streamlit  

## 🏗️ System Architecture  
The system collects data through IoT sensors connected to ESP32, processes it using Python, applies anomaly detection, and displays results via a Streamlit dashboard.
